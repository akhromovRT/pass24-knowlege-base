## Целевая аудитория

**Для кого:** Инженеры, технические специалисты, выполняющие первичное обследование объектов для оценки состояния установленного оборудования и возможности технического обслуживания

**Уровень подготовки:** Средний / Продвинутый

**Когда использовать:** При проведении первичного обследования объекта для оценки состояния установленных систем безопасности (СКУД, СРН, СОТ и пр.), определения необходимости технического обслуживания, замены или доукомплектации оборудования

---

## Краткое описание

Данный документ представляет собой форму (лист) первичного обследования объекта, предназначенную для систематизации информации о состоянии установленного оборудования на объекте заказчика. Лист используется при первичном обследовании объекта для сбора информации о системах, установленном оборудовании, его состоянии и составления рекомендаций по техническому обслуживанию, замене или доукомплектации составных частей систем.

**Важно:** Лист первичного обследования объекта заполняется инженером непосредственно на объекте во время проведения обследования и служит основой для формирования рекомендаций по техническому обслуживанию и дальнейшей работы с объектом.

---

## Предварительные требования

**Необходимые знания:**
- Понимание работы систем безопасности (СКУД, СРН, СОТ)
- Знание типов оборудования и его характеристик
- Умение оценивать состояние оборудования
- Навыки работы с технической документацией

**Необходимый доступ:**
- Доступ на объект заказчика
- Возможность осмотра установленного оборудования
- Доступ к технической документации на оборудование (если доступна)

**Необходимые материалы/ресурсы:**
- Бланк листа первичного обследования объекта (данный документ)
- Фотоаппарат или смартфон для фотографирования оборудования
- Инструменты для проверки состояния оборудования (при необходимости)
- Техническая документация на оборудование (если доступна)

## Основной контент

### Назначение листа первичного обследования объекта

Лист первичного обследования объекта предназначен для:

- Систематизации информации об объекте и установленном на нем оборудовании
- Оценки общего состояния оборудования
- Фиксации состава установленного оборудования с указанием типов, марок, моделей и количества
- Оценки состояния каждого элемента оборудования
- Формирования рекомендаций по техническому обслуживанию, замене или доукомплектации оборудования
- Документирования результатов первичного обследования для дальнейшей работы

**Важно:** Лист первичного обследования заполняется непосредственно на объекте во время проведения обследования. Все данные должны быть заполнены максимально точно и полно для обеспечения качества дальнейшей работы с объектом.

## Форма листа первичного обследования объекта

### Общая информация об объекте

**Название объекта:**  
_Заполнить полное название объекта (например: "Жилой комплекс 'Солнечный'")_

**Адрес объекта:**  
_Заполнить полный адрес объекта с указанием города, улицы, номера дома и других необходимых деталей_

**Ф.И.О., контактный номер телефона, электронная почта представителя заказчика, с которым осуществляется взаимодействие:**  
_Заполнить контактные данные представителя заказчика:_
- Фамилия, Имя, Отчество
- Контактный номер телефона
- Электронная почта

---

### Информация об установленных системах

**Системы, установленные на объекте:** (СКУД, СРН, СОТ и пр.)

_Указать все системы безопасности, установленные на объекте:_
- [ ] СКУД (Система контроля и управления доступом)
- [ ] СРН (Система распознавания номеров)
- [ ] СОТ (Система охранного телевидения / Видеонаблюдение)
- [ ] Другие системы: _________________________________

**Общее состояние оборудования:**

_Оценить общее состояние установленного оборудования:_
- [ ] Отличное — оборудование в рабочем состоянии, визуально в хорошем состоянии
- [ ] Хорошее — оборудование в рабочем состоянии, есть незначительные дефекты
- [ ] Удовлетворительное — оборудование работает, но требует технического обслуживания
- [ ] Неудовлетворительное — оборудование частично не работает, требуется ремонт или замена
- [ ] Критическое — оборудование не работает, требуется срочная замена

**Примечания инженера:** (рекомендации по замене или доукомплектовании составных частей систем)

_Заполнить рекомендации инженера по результатам обследования:_

```
[Здесь указываются рекомендации по:
- Замене неисправного или устаревшего оборудования
- Доукомплектовании систем недостающим оборудованием
- Техническому обслуживанию оборудования
- Улучшению работы систем
- Другим важным замечаниям и рекомендациям]
```

## Инструкция по заполнению листа

### Подготовка к заполнению

1. Перед выездом на объект подготовить бланк листа первичного обследования объекта
2. Уточнить у менеджера информацию об объекте (название, адрес, контактное лицо)
3. Подготовить инструменты для обследования (фотоаппарат, фонарик, мультиметр при необходимости)

### Заполнение на объекте

#### Шаг 1: Заполнение общей информации

**Что делать:**
1. Уточнить у представителя заказчика точное название объекта
2. Записать полный адрес объекта
3. Получить и записать контактные данные представителя заказчика:
   - ФИО
   - Номер телефона
   - Электронную почту

**Где:** На объекте, при встрече с представителем заказчика

**Результат:** Заполнена общая информация об объекте и контактах

---

#### Шаг 2: Определение установленных систем

**Что делать:**
1. Осмотреть объект и определить, какие системы безопасности установлены
2. Отметить в чек-листе установленные системы:
   - СКУД (контроллеры, считыватели, исполнительные устройства)
   - СРН (камеры распознавания номеров, серверы)
   - СОТ (камеры видеонаблюдения, видеорегистраторы)
   - Другие системы
3. Уточнить у представителя заказчика информацию о системах, если не очевидно

**Где:** На объекте, при осмотре установленного оборудования

**Результат:** Определены все системы безопасности, установленные на объекте

---

#### Шаг 3: Оценка общего состояния оборудования

**Что делать:**
1. Провести визуальный осмотр всего установленного оборудования
2. Проверить работоспособность ключевых элементов систем
3. Оценить общее состояние оборудования по критериям:
   - Отличное — все работает, визуально в хорошем состоянии
   - Хорошее — работает, есть незначительные дефекты
   - Удовлетворительное — работает, но требует обслуживания
   - Неудовлетворительное — частично не работает, требуется ремонт
   - Критическое — не работает, требуется срочная замена
4. Отметить соответствующую оценку в листе

**Где:** На объекте, при осмотре оборудования

**Результат:** Оценено общее состояние оборудования

---

#### Шаг 4: Заполнение таблицы состава оборудования

**Что делать:**
1. Для каждого элемента оборудования заполнить строку в таблице:
   - Определить тип оборудования
   - Записать марку (производителя)
   - Записать модель оборудования
   - Подсчитать количество единиц
   - Оценить состояние каждого элемента
   - Добавить примечания (место установки, особенности, проблемы)
2. При необходимости сделать фотографии оборудования для документации
3. Заполнить все обнаруженное оборудование

**Где:** На объекте, при детальном осмотре каждого элемента оборудования

**Результат:** Заполнена таблица состава оборудования с полной информацией о каждом элементе

**Важно:** Необходимо заполнить информацию о всех элементах оборудования, установленных на объекте. При большом количестве оборудования можно использовать дополнительные листы или электронную версию таблицы.

---

#### Шаг 5: Формирование рекомендаций

**Что делать:**
1. На основе оценки состояния оборудования сформировать рекомендации:
   - По замене неисправного или устаревшего оборудования
   - По доукомплектовании систем недостающим оборудованием
   - По техническому обслуживанию оборудования
   - По улучшению работы систем
2. Записать рекомендации в раздел "Примечания инженера"
3. Указать приоритетность рекомендаций (если применимо)

**Где:** На объекте или после обследования, при анализе собранной информации

**Результат:** Сформированы рекомендации по техническому обслуживанию, замене или доукомплектовании оборудования

---

#### Шаг 6: Завершение заполнения листа

**Что делать:**
1. Указать дату проведения обследования
2. Указать ФИО сотрудника, проводившего обследование
3. Поставить подпись
4. При необходимости получить подпись представителя заказчика (если требуется)

**Где:** На объекте, после завершения обследования

**Результат:** Лист первичного обследования объекта полностью заполнен и подписан

## Критерии успеха

**Успешное заполнение листа первичного обследования определяется следующими критериями:**

- Вся общая информация об объекте заполнена полностью и точно
- Определены все системы безопасности, установленные на объекте
- Оценено общее состояние оборудования
- Заполнена таблица состава оборудования для всех элементов, установленных на объекте
- Для каждого элемента указаны тип, марка, модель, количество и состояние
- Сформированы рекомендации по техническому обслуживанию, замене или доукомплектовании оборудования
- Лист подписан и датирован
- Информация достаточна для формирования плана технического обслуживания или дальнейшей работы с объектом

## Типичные ошибки и их решение

| Ошибка | Причина | Решение |
|--------|---------|---------|
| Неполная информация об оборудовании | Недостаточный осмотр объекта | Провести детальный осмотр всех помещений и зон объекта |
| Неверно указаны марка или модель оборудования | Отсутствие доступа к оборудованию или табличкам | Сделать фотографии оборудования, уточнить у представителя заказчика |
| Не оценено состояние оборудования | Недостаточная проверка работоспособности | Проверить работоспособность ключевых элементов, визуально оценить состояние |
| Отсутствуют рекомендации | Недостаточный анализ состояния | Проанализировать состояние каждого элемента и сформировать рекомендации |
| Не указаны контактные данные | Забыли уточнить у представителя | Обязательно получить контактные данные при встрече с представителем |
