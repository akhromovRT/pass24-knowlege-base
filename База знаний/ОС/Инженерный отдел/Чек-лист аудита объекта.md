## Целевая аудитория

**Для кого:** Инженеры, технические специалисты, выполняющие аудит объектов для оценки возможности внедрения систем PASS24, формирования технического задания и подбора оборудования

**Уровень подготовки:** Средний / Продвинутый

**Когда использовать:** При проведении аудита объекта для оценки технической возможности внедрения системы PASS24, сбора информации для формирования технического задания, определения мест размещения оборудования и оценки инфраструктуры объекта

---

## Краткое описание

Данный документ представляет собой регламент и чек-лист для проведения аудита объекта при работе с клиентами компании PASS24. Чек-лист содержит обязательные пункты для сбора информации об объекте, существующем оборудовании, инфраструктуре, требованиях заказчика и условиях работы системы контроля доступа. Документ обеспечивает единообразие процесса аудита и полноту собираемой информации, необходимой для формирования технического задания и подбора оборудования.

**Важно:** Чек-лист должен заполняться инженером непосредственно на объекте во время проведения аудита. Все пункты должны быть заполнены максимально точно и полно, с фотографированием и схематическим указанием ключевых элементов объекта.

## Предварительные требования

**Необходимые знания:**
- Понимание работы систем PASS24 (PASS24.online, PASS24.cloud, PASS24.auto)
- Знание типовых схем реализации систем контроля доступа
- Понимание принципов работы СКУД и СРН
- Знание технических характеристик оборудования (камеры, шлагбаумы, контроллеры, считыватели)
- Умение оценивать технические возможности интеграции оборудования
- Навыки работы с технической документацией

**Необходимый доступ:**
- Доступ на объект заказчика
- Возможность осмотра объекта и существующего оборудования
- Доступ к технической документации на существующее оборудование (если доступна)

**Необходимые материалы/ресурсы:**
- Бланк чек-листа аудита объекта (данный документ)
- Фотоаппарат или смартфон для фотографирования объекта
- Рулетка или лазерный дальномер для замеров
- Планшет или ноутбук для заполнения чек-листа
- Шаблоны технических заданий
- Информация о типовых схемах реализации различных задач

## Связь с общей последовательностью работ

Данный чек-лист соответствует **Регламенту обследования объекта** и используется на этапе **проведения обследования на объекте** (Шаги 3-6 регламента):

- **Шаг 3:** Встреча с заказчиком и начало обследования — заполнение раздела "Контактная информация"
- **Шаг 4:** Обследование инфраструктуры объекта — заполнение разделов "Описание объекта", "Инфраструктура объекта", "Места размещения оборудования"
- **Шаг 5:** Консультация и презентация — уточнение требований заказчика, определение типа кодоносителей
- **Шаг 6:** Заполнение чек-листа — систематизация всей собранной информации

Результаты заполнения чек-листа используются на этапе **обработки результатов обследования** (Шаги 7-9 регламента) для формирования схемы объекта, концепта оборудования и технического задания.

## Рекомендации по использованию чек-листа

### Соответствие техническим требованиям систем PASS24

При заполнении чек-листа необходимо учитывать технические требования к оборудованию систем PASS24:

**Для систем распознавания номеров (PASS24.auto):**
- Камеры должны обеспечивать разрешение минимум Full HD (1920x1080), рекомендуется 2MP и выше
- Обязательна поддержка ИК-подсветки для работы в темное время суток
- Защита от погодных условий: IP66 и выше
- Оптимальное размещение: перед шлагбаумом на расстоянии 3-5 метров, высота 2-3 метра
- Рекомендуется установка 2 камер на полосу движения для повышения точности распознавания

**Для систем контроля доступа (СКУД):**
- Контроллеры должны поддерживать протоколы связи, совместимые с PASS24.online
- Считыватели должны поддерживать выбранный тип кодоносителей (BLE/NFC для PASS24.Key, карты для традиционных систем)
- Исполнительные устройства (шлагбаумы, турникеты) должны иметь возможность интеграции через контроллеры доступа

**Для инфраструктуры:**
- Стабильное интернет-соединение обязательно для работы облачных систем PASS24
- Электроснабжение 220V должно обеспечивать достаточную мощность для всего оборудования
- Коммутационные шкафы должны обеспечивать защиту оборудования от внешних воздействий

### Порядок заполнения чек-листа

1. **Подготовительный этап:** Изучить информацию о клиенте, предоставленную менеджером, определить тип ситуации согласно Регламенту обследования объекта
2. **На объекте:** Заполнять чек-лист последовательно, не пропуская разделы
3. **Фотографирование:** Делать фотографии всех ключевых элементов объекта параллельно с заполнением соответствующих разделов
4. **Схематирование:** Создавать схему объекта с указанием мест размещения оборудования по мере заполнения разделов
5. **Проверка:** После завершения аудита проверить полноту заполнения всех обязательных пунктов

### Дополнительные рекомендации

- При наличии существующего оборудования обязательно указать его характеристики и возможность интеграции
- Учитывать перспективы развития объекта (планируемый демонтаж, строительство и т.д.)
- Обращать внимание на факторы, влияющие на точность распознавания номеров (освещение, препятствия, угол установки камер)
- Учитывать требования безопасности заказчика при выборе типа кодоносителей
- При сложных условиях на объекте делать дополнительные фотографии и замеры
