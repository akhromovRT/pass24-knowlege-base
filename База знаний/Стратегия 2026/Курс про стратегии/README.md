# Курс про стратегию

## Описание раздела

Данный раздел содержит материалы курса по разработке стратегии компании, основанного на практическом опыте проведения стратегических сессий.

## Содержимое раздела

### Документы

- **[Содержание курса по стратегии](./Содержание курса по стратегии.md)** — краткое описание содержания курса на основе стенографии лекции Павла Ланенкова. Содержит ключевые принципы стратегии, методы разработки и реализации, практические рекомендации.

### Материалы

- **Курс_Long_Разработка_стратегии_компании_01_2026.pdf** — презентация курса

## Назначение материалов

Материалы курса предназначены для:
- Собственников бизнеса и топ-менеджеров
- Предпринимателей, планирующих стратегическое развитие компании
- Команд, готовящихся к проведению стратегических сессий
