# Инструкции PASS24.control

## Описание раздела

Инструкции по программному обеспечению PASS24.control и оборудованию (в т.ч. PASS24-400): веб-интерфейс, IP-клиент, настройка QR и пин-кода, e-mail оповещения, BLE-ключи, шифрование считывателей.
