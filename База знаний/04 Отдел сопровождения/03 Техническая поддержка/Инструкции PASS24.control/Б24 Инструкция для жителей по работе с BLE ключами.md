## Основной контент

Инструкция для жителей по работе с BLE ключами 
Примечание: для получения ключа обратитесь в управляющую компанию 
Вашего посёлка. 
Приложение доступно для установки на мобильные операционные системы 
iOS и Android. Требования к версии операционной системы указаны на 
страницах приложений. 
Для установки приложения на свой смартфон, найдите приложение 
PASS24.key в App Store для телефонов iPhone и Google Play для устройств 
на Android, или отсканируйте QR-код ниже. 
 
После установки приложения, вы получите уведомления о необходимости 
предоставить приложению разрешение на использование функций телефона 
“Bluetooth” и “Геопозиция”. Необходимо предоставить свое согласие. Без 
подключения этих функций, использование приложения невозможно. На 
устройствах Android согласие может запрашиваться позднее. 
 
Обратите внимание, что приложение никуда не передает данные о вашем 
местоположении и не занимается отслеживанием ваших передвижений. 
Функция геолокация используется только для корректной работы 
передатчика . 
 
Для входа в приложение используйте данные Вашего аккаунта 
PASS24.online или нажмите “Войти по SMS” в нижней части экрана. 
 
Далее описана процедура установки на примере устройств Android. Для 
иных производителей последовательность действий будет аналогичная, 
отличаются только внешний вид некоторых элементов приложения.
