## Типичные запросы и ответы

### 1. Не приходит SMS-код для регистрации

**Типичная формулировка клиента:**
- "Не приходит смс для регистрации"
- "Не приходит смс код на номер телефона"
- "Не могу зарегистрироваться, не приходит код"
- "Проблема носит массовый характер, не приходит смс"

**Стандартный ответ:**

```
Здравствуйте!

Благодарим за обращение. Понимаем, что отсутствие SMS-кода препятствует регистрации в приложении.

Для решения проблемы просим выполнить следующие действия:

1. Проверьте, что номер телефона введен корректно (в формате +7XXXXXXXXXX)
2. Убедитесь, что на телефоне нет блокировки SMS от коротких номеров
3. Проверьте папку "Спам" в сообщениях
4. Подождите 2-3 минуты - иногда SMS приходит с задержкой
5. Попробуйте запросить код повторно через 1-2 минуты

Если проблема сохраняется, пожалуйста, укажите:
- Номер телефона (в формате +7XXXXXXXXXX)
- Модель телефона и версию ОС
- Название оператора связи (МТС, Билайн, Мегафон, Теле2 и т.д.)
- Время, когда вы пытались получить код

Мы проверим возможность отправки SMS на ваш номер и при необходимости восстановим доступ вручную.

С уважением,
Служба технической поддержки PASS24.online
```

**Диагностика:**
1. Проверить в системе, был ли отправлен SMS-код
2. Проверить статус номера телефона в базе данных
3. Проверить наличие блокировок на номере
4. При необходимости отправить код вручную через админ-панель

**Возможность автоматизации:** Высокая (80%)
- Автоматическая проверка статуса отправки SMS
- Автоматическая повторная отправка при сбое
- Автоматическая проверка блокировок номера

- "Пытаюсь восстановить пароль, запрашиваю смс с кодом. СМС не приходит"

**Стандартный ответ:**

```
Здравствуйте!

Благодарим за обращение. Понимаем важность восстановления доступа к приложению.

Для решения проблемы с восстановлением пароля просим выполнить следующие действия:

1. Убедитесь, что вы используете тот же номер телефона, который был указан при регистрации
2. Проверьте, что номер телефона введен в правильном формате (+7XXXXXXXXXX)
3. Проверьте папку "Спам" в сообщениях
4. Подождите 2-3 минуты - иногда SMS приходит с задержкой
5. Попробуйте запросить код повторно через 1-2 минуты

Если проблема сохраняется, пожалуйста, укажите:

- Модель телефона и версию ОС
- Название оператора связи
- Примерное время последней успешной авторизации (если помните)

Мы проверим ваш аккаунт в системе и при необходимости восстановим доступ вручную.

С уважением,
Служба технической поддержки PASS24.online
```

**Диагностика:**
1. Проверить наличие аккаунта с указанным номером телефона
2. Проверить статус отправки SMS для восстановления пароля
3. Проверить наличие блокировок на номере
4. При необходимости сбросить пароль вручную через админ-панель

**Возможность автоматизации:** Высокая (85%)
- Автоматическая проверка существования аккаунта
- Автоматическая отправка SMS-кода
- Автоматическая проверка блокировок

### 3. Не могу зарегистрироваться - номер уже существует в системе

**Типичная формулировка клиента:**
- "Не могу зарегистрироваться, пишет что в системе уже есть такой номер"
- "Пишет, что уже есть такой номер в системе. Не дает пройти регистрацию дальше"
- "Не могу зарегистрироваться как новый пользователь. Пишет что в системе уже есть такой номер"

**Стандартный ответ:**

```
Здравствуйте!

Благодарим за обращение. Понимаем, что проблема с регистрацией препятствует использованию приложения.

Если при регистрации появляется сообщение о том, что номер уже существует в системе, это означает, что аккаунт с таким номером телефона уже был создан ранее.

Возможные причины:
1. Вы уже регистрировались ранее, но забыли об этом
2. Номер был зарегистрирован администратором объекта
3. Номер был зарегистрирован другим пользователем (например, предыдущим владельцем квартиры)

Рекомендуемые действия:

1. Попробуйте войти в приложение, используя функцию "Забыли пароль?"
2. Если восстановление пароля не работает, укажите:
   - Номер телефона
   - Название объекта (ЖК, КП и т.д.)
   - Номер квартиры/участка (если применимо)
   - Ваши ФИО

Мы проверим информацию в системе и при необходимости:
- Восстановим доступ к существующему аккаунту
- Создадим новый аккаунт, если номер был зарегистрирован ошибочно
- Свяжем ваш номер с правильным объектом

С уважением,
Служба технической поддержки PASS24.online
```

**Диагностика:**
1. Проверить наличие аккаунта с указанным номером
2. Проверить, к какому объекту привязан аккаунт
3. Проверить статус аккаунта (активен/заблокирован)
4. При необходимости перепривязать номер к правильному объекту или создать новый аккаунт

**Возможность автоматизации:** Средняя (60%)
- Автоматическая проверка существования аккаунта
- Автоматическое предложение восстановить пароль
- Ручная проверка требуется для перепривязки к объекту

С уважением,

- Автоматическое предложение восстановить пароль

- Ручное вмешательство требуется только в сложных случаях

### 5. Приложение вылетает / разлогинилось

**Типичная формулировка клиента:**
- "Вылетело приложение"
- "Разлогинилось из приложения"
- "Приложение закрывается при попытке входа"
- "Слетел пароль, не могу войти"

**Стандартный ответ:**

```
Здравствуйте!

Благодарим за обращение. Понимаем, что проблемы с приложением мешают использованию сервиса.

Если приложение вылетает или вы разлогинились, рекомендуем выполнить следующие действия:

1. Перезагрузите устройство
2. Обновите приложение до последней версии (App Store / Google Play)
3. Удалите и заново установите приложение
4. Очистите кэш приложения (если доступно)
5. Попробуйте войти заново, используя номер телефона и пароль

Если проблема сохраняется, пожалуйста, укажите:
- Модель телефона и версию ОС
- Версию приложения (можно найти в настройках приложения)
- Описание проблемы (когда вылетает, при каких действиях)
- Номер телефона, с которым возникла проблема

Мы проверим информацию и при необходимости восстановим доступ к аккаунту.

С уважением,
Служба технической поддержки PASS24.online
```

**Диагностика:**
1. Проверить статус аккаунта в системе
2. Проверить версию приложения клиента
3. Проверить наличие известных проблем с версией приложения
4. При необходимости восстановить сессию или сбросить пароль

**Возможность автоматизации:** Средняя (50%)
- Автоматическая проверка версии приложения
- Автоматическое предложение обновить приложение
- Ручная диагностика требуется для сложных случаев

## Рекомендации по автоматизации

### Высокий приоритет автоматизации

1. **Автоматическая отправка SMS-кодов** (85% запросов)
   - Автоматическая повторная отправка при сбое
   - Автоматическая проверка статуса доставки
   - Автоматическое уведомление клиента о проблемах с доставкой

2. **Автоматическое восстановление пароля** (90% запросов)
   - Автоматическая проверка существования аккаунта
   - Автоматическая отправка SMS-кода для восстановления
   - Автоматическое предложение восстановить пароль при ошибке входа

3. **Автоматическая проверка номера при регистрации** (60% запросов)
   - Автоматическое предложение восстановить пароль, если номер существует
   - Автоматическая проверка привязки номера к объекту

### Средний приоритет автоматизации

1. **Автоматическая диагностика проблем с приложением** (50% запросов)
   - Автоматическая проверка версии приложения
   - Автоматическое предложение обновить приложение
   - Автоматическая проверка известных проблем

### Ожидаемый эффект от автоматизации

- **Сокращение времени обработки:** с 15-30 минут до 2-5 минут для типичных случаев
- **Снижение нагрузки на менеджеров:** до 70% запросов могут быть обработаны автоматически
- **Улучшение качества обслуживания:** единообразные ответы, быстрая реакция
