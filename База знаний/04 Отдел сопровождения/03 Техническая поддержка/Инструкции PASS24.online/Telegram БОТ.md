## Основной контент

Инструкция по использованию Telegram-бота для заказа 
 разового пропуска 
 1. Найдите бота в Telegram, введите его имя в строке поиска @PASS24ROBOT и 
 нажмите "Start". 
 2. Бот приветствует вас и предлагает отправить номер телефона для подтверждения. 
 Нажмите на кнопку "Подтвердить номер телефона" 
 Примечание: Для заказа пропусков через БОТ нужно установить никнейм в 
 профиле Telegram. 

 3. После подтверждения номера телефона, появится главное меню бота. Выберите 
 опцию "Разовый". 
 4. Следующим шагом выберите объект, для которого вы хотите заказать пропуск. 
 Бот предоставит вам список доступных Вам объектов. 
 5. После выбора объекта, выберите адрес, по которому будет действовать пропуск. 
 6. Затем выберите тип пропуска: "Автомобиль" или "Пешеход". Если вам нужен 
 пропуск для автомобиля,выберите тип транспорта и укажите его регистрационный 
 номер. 

 7. После выбора типа пропуска "Пешеход", вам будет предложено заполнить данные 
 пропуска(обычно это ФИО посетителя). 
 8. После заполнения всех данных, бот попросит вас подтвердить заказ пропуска. 
 Проверьте все введенные данные и нажмите кнопку "Создать" 
 9. После подтверждения заказа, бот сообщит вам об успешном создании пропуска и 
 предоставит ссылку, которой можно поделиться с посетителем. 
 Примечание: заказывать пропуск через Telegram-бот может только аторизованный 
 пользователь системы PASS24.online
