## О компании PASS24.online

**PASS24.online** — это облачная система (SaaS) для автоматизации управления доступом на территорию жилых комплексов, коттеджных поселков, бизнес-центров и промышленных объектов.

### Разработчик
- **Компания:** ОНВИ Сервис (входит в группу LANIT)
- **Год основания:** 2016
- **Модель работы:** SaaS (Software as a Service)
- **Инфраструктура:** Облачная платформа OnCloud.ru (совместный проект с компанией «Онланта»)

### Назначение системы
- Автоматизация выдачи пропусков
- Замена бумажных журналов на электронные архивы
- Управление доступом на объекты
- Аналитика посещаемости и событий
- Интеграция с существующими системами СКУД

### Преимущества решения
- **Быстрое развертывание:** система настраивается примерно за 1 час
- **Гибкость:** может работать как самостоятельная система или как надстройка над существующей СКУД
- **Экономия:** сокращение расходов на физическую охрану и бумажные журналы
- **Безопасность:** облачное хранение данных с высокой отказоустойчивостью
- **Масштабируемость:** без ограничений по числу сотрудников, пользователей и пропусков

## Продуктовая линейка

### 1. PASS24.online (Базовая платформа)

**Описание:** Облачная платформа для автоматизации управления доступом и выдачи пропусков.

**Основные возможности:**
- Мобильное бюро пропусков для дистанционной выдачи временных и постоянных пропусков
- Электронный журнал событий в облаке
- Веб-интерфейсы для разных групп пользователей:
  - Мобильное приложение для жителей/арендаторов
  - Веб-интерфейс для охраны
  - Панель администратора для управляющих компаний
- Мгновенные push-уведомления о прибытии гостей
- История всех запрошенных пропусков и визитов
- Аналитика посещаемости в режиме реального времени

**Способы идентификации:**
- QR-коды
- NFC (Near Field Communication)
- Face ID (распознавание лица)
- Bluetooth (BLE) через мобильное приложение

**Интеграция с голосовыми помощниками:**
- Заказ пропусков голосом через Яндекс Алису

**Платформы:**
- iOS
- Android
- Web-интерфейс

---

### 2. PASS24.Key

**Описание:** Мобильное приложение, которое превращает смартфон в универсальный ключ доступа.

**Основные возможности:**
- Использование смартфона вместо физических карт доступа
- Управление доступом через Bluetooth (BLE-идентификатор)
- Контроль различных систем доступа:
  - Турникеты
  - Домофоны
  - Шлагбаумы
  - Лифты

**Технологии:**
- BLE (Bluetooth Low Energy)
- NFC
- QR-коды

**Преимущества:**
- Не нужно носить физические карты или ключи
- Универсальное решение для всех типов доступа
- Удобное управление через мобильное приложение

---

### 3. PASS24.auto

**Описание:** Программно-аппаратный комплекс для автоматизации проезда автотранспорта через шлагбаумы с использованием распознавания автомобильных номеров.

**Основные возможности:**
- Автоматическое распознавание автомобильных номеров
- Автоматическое открытие шлагбаумов для авторизованного транспорта
- Интеграция с системами видеонаблюдения
- Ведение журнала проездов транспорта
- Аналитика автомобильного трафика

**Интеграции:**
- CVS (система распознавания номеров)
- Trassir (видеонаблюдение)
- Ivideon (видеонаблюдение)
- «Автомаршал» (распознавание номеров)

**Применение:**
- Управление парковками
- Контроль проезда на территорию объектов
- Автоматизация КПП (контрольно-пропускных пунктов)

---

### 4. PASS24.admin

**Описание:** Веб-интерфейс для администраторов, управляющих компаний и службы охраны.

**Основные возможности:**
- Управление правами доступа
- Мониторинг событий в режиме реального времени
- Настройка правил доступа
- Управление пользователями и пропусками
- Формирование отчетов и аналитики
- Работа с электронным журналом событий

**Целевая аудитория:**
- Администраторы управляющих компаний
- Служба охраны
- Менеджеры по безопасности объектов

## Тарифные планы

Система PASS24.online предлагает четыре тарифных плана с различным функционалом. Конкретная стоимость тарифов предоставляется по запросу или через калькулятор решений на официальном сайте.

### Тариф «Стандарт»

**Описание:** Базовый тариф для автоматизации заказа пропусков и ведения электронного журнала.

**Включает:**
- Базовую автоматизацию заказа пропусков
- Электронный журнал в облаке (замена бумажных журналов)
- Мобильное приложение для жителей/арендаторов
- Веб-интерфейс для охраны
- Панель администратора
- Без ограничений по числу сотрудников
- Без ограничений по числу пользователей
- Без ограничений по количеству пропусков

**Подходит для:**
- Объектов, которым нужна базовая автоматизация пропускного режима
- Замены бумажных журналов на электронные
- Упрощения процесса заказа пропусков

---

### Тариф «Авто»

**Описание:** Тариф для управления парковкой и автоматизации проезда транспорта.

**Включает:**
- Все функции тарифа «Стандарт»
- Подключение 2 камер для видеораспознавания автомобильных номеров
- Автоматическое открытие шлагбаумов для авторизованного транспорта
- Интеграция с системами распознавания номеров (CVS, Trassir, Ivideon, «Автомаршал»)
- Ведение журнала проездов транспорта

**Подходит для:**
- Объектов с парковками
- Бизнес-центров и складских комплексов
- Коттеджных поселков с контролем проезда транспорта

---

### Тариф «СКУД»

**Описание:** Тариф для реализации полноценной пропускной системы с современными способами идентификации.

**Включает:**
- Все функции тарифа «Стандарт»
- Мобильная идентификация (PASS24.Key)
- Использование QR-кодов для доступа
- Поддержка ключей доступа (карты и BLE)
- Управление доступом через турникеты, домофоны, шлагбаумы, лифты

**Подходит для:**
- Объектов, требующих контроля пешеходного трафика
- Интеграции с существующими системами СКУД
- Объектов с несколькими точками доступа

---

### Тариф «Авто + СКУД»

**Описание:** Максимальный набор функций для комплексного управления как автомобильным, так и пешеходным трафиком.

**Включает:**
- Все функции тарифа «Авто»
- Все функции тарифа «СКУД»
- API для интеграции с любыми сторонними системами СКУД
- API для интеграции с CRM-системами
- Подключение 2 камер для распознавания номеров
- Глубокая интеграция с оборудованием (Sigur, Trassir и др.)

**Подходит для:**
- Крупных объектов с комплексными требованиями к безопасности
- Объектов, требующих интеграции с существующими системами
- Предприятий с высокими требованиями к автоматизации

## Услуги и интеграции

### Интеграции с системами СКУД

PASS24.online может работать как самостоятельная система или как надстройка над существующей СКУД объекта.

**Поддерживаемые системы СКУД:**
- **Sigur** — российская система контроля и управления доступом
- **Bolid** — системы безопасности и автоматизации
- Другие системы через API

**Возможности интеграции:**
- Синхронизация данных между системами
- Единое управление доступом
- Совместное использование оборудования

---

### Интеграции с системами видеонаблюдения

**Поддерживаемые системы:**
- **Trassir** — профессиональное ПО для видеонаблюдения
- **Ivideon** — облачное видеонаблюдение
- **CVS** — система распознавания номеров
- **«Автомаршал»** — система распознавания автомобильных номеров

**Возможности интеграции:**
- Автоматическое открытие шлагбаумов при распознавании номеров
- Связывание событий доступа с видеоархивом
- Настройка реакций камер на события СКУД
- Видеоверификация проходов/проездов

---

### Интеграции с голосовыми помощниками

**Яндекс Алиса:**
- Заказ пропусков голосовыми командами
- Управление доступом через голосовой интерфейс
- Получение информации о статусе пропусков

---

### API и технические интеграции

**Возможности API:**
- Интеграция с любыми сторонними системами СКУД
- Интеграция с CRM-системами
- Синхронизация данных с внешними системами
- Автоматизация бизнес-процессов

**Доступность API:**
- Доступно в тарифе «Авто + СКУД»
- Предоставляется для глубокой интеграции с существующими системами
