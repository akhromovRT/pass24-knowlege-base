## Основной контент

Шифрование считывателя 
 
Скачайте и установите мобильное программное обеспечение PW Config. 
Запустите PW-Config, нажмите кнопку "Поиск", начнется поиск устройств.После сканировании 
выберите считыватель из списка и нажмите кнопку «Подключиться», введите код инженера - 
будет выполнена вычитка конфигурации. 
 
После успешной вычитки будет доступно основное меню, с помощью которого можно 
настроить шифрование считывателя. 
Выберите Профили Mifare → Mifare Classic → Безопасность → SL3 диверсификация → 
 

 
Ключ (впишите ключ шифрования 32 символа) → Использовать сектор (выберите нужный 
сектор) → Читать (выберите Код карты или По адресу) → Использовать адрес (если выбрали 
читать по адресу, то выберите какой адрес использовать) → Порядок байт кода карты 
(выберите Прямой) 
 
После настройки шифрования считывателя в главном меню станет доступен пункт "Записать в 
устройство". По его нажатию конфигурация будет записана в устройство
