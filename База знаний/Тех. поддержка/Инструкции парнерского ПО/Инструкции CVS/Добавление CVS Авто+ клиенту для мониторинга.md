## Основной контент

Как добавить клиентскую программу CVS Авто+ для мониторинга на экран 
охраны 
1. Скачиваем CVS Авто+ с официального сайта ООО Новые технологии 
2. Запускаем от имени администратора, устанавливаем 
3. Для входа вводим пароль admin 
4. Заходим в настройки: 
 - Общее – выбираем кол-во камер и с какой камеры начинать показ. Внизу ставим «Сетевой 
клиент для мониторинга» 
 - Видеосерверы – вводим IP-адрес сервера, с какого передавать картинку 
 - База данных – вводим этот же IP-адрес 
5. Жмем «ОК». 
6. На окне камеры выбираем нужную камеру. Должно вывести изображение. 
7. Сохраняем программу в автозагрузку. 
Добавляем программу CVS Авто+ в автозагрузку: 
1. Win+R – вводим «regedit» 
2. В окне редактора реестра переходим по иерархии: HKEY_CURRENT_USER — Software — 
Microsoft — Windows — Current Version — Run и, щелкнув в свободном месте окна правой 
кнопкой мыши, выбираем команду «Создать» — «Строковый параметр». 
3. Вводим название программы и нажимаем «Enter». Затем дважды щелкаем мышью по новому 
параметру и в строке «Значение» вводим путь к нужному файлу и нажимаем «ОК».
