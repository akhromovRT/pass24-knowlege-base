# Инструкции PASS24.online

## Описание раздела

Инструкции по платформе PASS24.online: мобильные приложения (Android, iOS), Telegram-бот, веб-интерфейс для администраторов, операторов и охраны, заказ пропусков, доверенности, интеграция с камерами, массовая рассылка.

## Видеоинструкции

- **Инструкции по WEB/** — модули веб-интерфейса: текстовые инструкции (см. таблицу выше), скриншоты и исходные видео (формат `-converted.mp4`)
- **Видео Инструкции Приложения/** — исходные видеофайлы с инструкциями по работе с мобильным приложением (пропуск пешеход/авто, приглашение, запросы на пропуска, доверенность, адрес, обращение в УК). Текстовые версии инструкций доступны в папке [Мобильное приложение](./Мобильное%20приложение/)
