## Основной контент

Инструкция для Администратора 
 Вход в систему 
 Для входа в систему PASS24.online: 
 - перейдите по ссылке https://hospital.pass24online.ru 
 - введите e-mail 
 - введите пароль 
 Нажмите «Войти» 
 Настройка прав доступа 
 Перейдите во вкладку Права в боковом меню находясь в данном разделе, 
 нажмите Создать подгруппу в группе Администратор. 
 1. Введите Имя группы 
 2. Нажмите Сохранить 

 Чтобы настроить полномочия - доступ к каким модулям могут иметь 
 сотрудники в системе, нажмите на шестеренку. 
 Впоявивщемся окне переходите по разделам(1) и задаваете уровень доступа 
 наследует/разрешен/запрещен (2), к той или иной функции приложения. 
 После того, как установите права доступа - нажмите на кнопку Сохранить 
 (3), чтобы сохранить настройки. 

 Создание сотрудника 
 Перейдите во вкладку Сотрудники в боковом меню и щелкните на кнопку 
 + НОВЫЙ СОТРУДНИК в правом верхнем углу для добавления нового 
 сотрудника. 
 1. Введите ФИО сотрудника 
 2. Введите E-mail сотрудника 
 3. Задайте Пароль 
 4. Введите Номер телефона сотрудника 
 5. Выберите Группу в соответствии с правами доступа сотрудника 
 6. Выберите Объект 
 7. Выберите КПП 
 8. Нажмите Сохранить 

 Создание адреса 
 Перейдите во вкладку Адреса в боковом меню и щелкните на кнопку 
 Добавить. 
 1. Выберите Объект 
 2. Заполните необходимые поля Адреса 
 3. Нажмите Сохранить 

 Создание пользователей 
 Перейдите во вкладку Пользователи в боковом меню и щелкните на кнопку 
 Добавить в правом верхнем углу для создания пользователя. 
 1. Введите ФИО 
 2. При необходимости введите Е-mail и Описание 
 3. Введите Номер телефона 
 4. Добавьте Адрес 
 5. Нажмите Сохранить 

 Создание пропуска 
 Перейдите во вкладку Пропуска в боковом меню и щелкните на кнопку 
 +НОВЫЙ ПРОПУСК в правом верхнем углу для создания пропуска. 
 1. Выберите Адрес 
 2. В Пользователях выберите себя 
 3. Выберите тип пропуска Разовый/Временный 
 4. Укажите Время визита 
 5. Выберите Автомобиль 
 6. Выберите Тип гостя 
 7. Введите Марку и Номер автомобиля 
 8. Введите Комментарий 
 9. Нажмите Сохранить 
 Пропуск успешно создан. 

 Импорт пользователей и пропусков 
 Для импорта Пользователей и Пропусков: 
 1. Подготовьте файл по образцу import-example.xlsx (путь к образцу 
 импорта C:\import) 
 В столбце Тип Автомобиля(Тип транспорта) нужно нужно указать ID: 
 ID Тип транспорта 
 1 Личный транспорт 
 2 Скорая помощь 
 3 Посетитель 
 4 Спец. Пропуск 
 2. Поместите файл в папку C:\import (наименование файла должно быть 
 import.xlsx) 
 3. Запустите файл start.bat в папке C:\import двойным нажатием левой 
 кнопки мыши 
 Примечание: Если в настройках объекта Временные пропуска - По запросу, 
 то все импортируемые пропуска попадут в раздел запросы. 
 Работа с запросами 
 Перейдите во вкладку Запросы в боковом меню и щелкните на входящий 
 запрос. 

 1. Нажмите на входящий запрос 
 2. Нажмите Одобрить 
 3. Подтвердите одобрение, нажав Одобрить 
 После одобрения запроса, пропуск автоматически появится в разделе 
 Пропуска. 
 Для отклонения запроса: 
 1. Нажмите Отклонить 
 2. Введите Причину отклонения 
 3. Подтвердите отклонение, нажав Отклонить 
 * причина отклонения обязательна для заполнения 

 Выгрузка отчёта 
 Для работы скрипта требуется: 
 1. Установить интерпретатор python 3 
 2. ПК с которого запускается скрипт должен быть с включенным OpenVPN 
 3. Установить необходимые библиотеки, для через командную строку 
 запустите команду pip install -r requirements.txt. 
 Скрипт запускается из командной строки: 
 python main.py { host } { database } { user } { password } {from } { to } —port 
 { port } 
 - name Имя скрипта 
 - host Хост базы данных 
 - database Имя базы данных 
 - user Имя пользователя БД 
 - password Пароль пользователя БД 
 - date_from Дата/время начала отчёта. Если время не указано, 
 автоматически берет время с 00:00:00 
 - date_to Дата/время окончания отчёта. Если время не указано, 
 автоматически берет время до 00:00:00 
 Порт опциональный, по-умолчанию 3306 
 После отработки скрипта в папке 
 “onvis_team-v2-report-creator-python-519e9b9bce9b” появится папка 
 “reports”, в которой будет файл отчёта. 
 Примечание: Вызов инструкции командой python main.py -h
