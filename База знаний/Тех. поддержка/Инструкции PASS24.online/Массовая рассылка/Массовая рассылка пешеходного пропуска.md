## Основной контент

Массовая рассылка пешеходного пропуска
Чтобы загрузить список пешеходных пропусков, переходим в модульПропуска и в правом верхнем углу нажимаем“+Новыйпропуск”1. Выбираем Адрес2. Выбираем Пользователя
Примечание: поля Адрес и Пользователь связаны между собой. Таким образом, выбрав адрес,в поле выбора пользователя будут показаны только те пользователи, которые имеют доступ квыбранному адресу. И наоборот - при выборе пользователя первым, в списке адресов будутдоступны только те адреса, к которым у пользователей есть доступ.
3. Выбираем Тип пропуска (разовый временный, постоянный)4. Указываем время визита5. Выбираем тип гостя Пешеход6. Нажимаем выбрать Файл
7. Выбираем заранее подготовленный файл8. Нажимаем Открыть

Список пропусков появиться в карточке создания пропусков
9. Нажмите Сохранить
Появиться сообщение об успешном сохранении пропуска.
Чтобы сделать массовую рассылку пропусков:1. Нажмите Режим отправки

2. Чек-боксами выберите нужные пропуска3. Нажмите Отправить выбранные адреса
4. Появиться сообщение, что пропуска отправляются
Правила заполнения файла импорта пешеходных пропусков
1. Загрузка только excel файлов2. В файле не должно быть заголовков столбцов, сразу строки
3. В файле должно быть только 2 столбца ФИО и E-mail
При импорте файла идёт проверка, если где то некорректно заполнен e-mailли не указаны ФИО, то покажет ошибку и номер строки.
