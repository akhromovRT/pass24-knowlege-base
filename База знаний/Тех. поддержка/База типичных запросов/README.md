## Краткое описание

База типичных запросов и ответов создана на основе анализа всех обращений в службу технической поддержки за 2025 год (3454 заявки). Содержит структурированные типичные запросы клиентов с готовыми ответами и рекомендациями по решению проблем. База знаний предназначена для ускорения обработки обращений и обеспечения единообразия ответов.

---

## Структура базы знаний

База знаний организована по категориям проблем:

### 1. [Регистрация и вход в приложение](./Регистрация%20и%20вход.md)
- Проблемы с получением SMS-кода
- Восстановление пароля
- Ошибки при входе
- Проблемы с регистрацией

### 2. [Распознавание номеров автомобилей](./Распознавание%20номеров.md)

- Проблемы с отдельными номерами

### 3. [Пропуска и шлагбаумы](./Пропуска%20и%20шлагбаумы.md)

### 4. [Проблемы с приложением](./Проблемы%20с%20приложением.md)

### 5. [Оборудование](./Оборудование.md)
- Проблемы с камерами
- Неисправности считывателей

### 6. [Доверенные лица](./Доверенные%20лица.md)

- Проблемы с добавлением

### 7. [Добавление объектов](./Добавление%20объектов.md)

## Как использовать базу знаний

1. **Определите категорию проблемы** клиента по описанию обращения
2. **Откройте соответствующий документ** из списка выше
3. **Найдите типичный запрос**, наиболее близкий к проблеме клиента
4. **Используйте готовый ответ** или адаптируйте его под конкретную ситуацию
5. **Следуйте рекомендациям** по диагностике и решению проблемы
