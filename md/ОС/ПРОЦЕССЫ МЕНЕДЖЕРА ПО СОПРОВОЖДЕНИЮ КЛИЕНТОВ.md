
ПРОЦЕССЫ МЕНЕДЖЕРА ПО СОПРОВОЖДЕНИЮ КЛИЕНТОВ

   1. Сопровождение клиентов по вопросам работы с нашим приложением и
      распределением по вопросам, которые относятся к другим отделам, таким
      образом менеджер:


   - принимает звонки, на техподдержку;
   - передаёт информацию в ТП;
   - согласовывает даты монтажа;
   - выявляет первичные потребности и желаний при знакомстве
   - выясняет, какое оборудование уже установлено;
   - уточняет даты готовности площадки к монтажу;
   - снимает критические разногласия и претензии со стороны клиента.


   2. Сбор абонентского платежа ежемесячно:


   - ежемесячно, начиная с 20-ого числа каждодневные звонки клиентам с
   напоминанием оплат;
   - дублирование счетов тем, кто потерял их в почте.


   3. Сопровождение и организация доставок:


   - отслеживание баланса на Достависте;
   - вызов транспорта для отгрузки;
   - мониторинг курьера и качества доставки;
   - поиск крупногабаритного транспорта для отгрузки шлагбаумов, турникетов
   и стрел.


   4. Контроль отгрузочных документов:


   - мониторинг отгрузок;
   - контроль подписания закрывающих документов.


   5. Контроль отгрузки товара клиенту:


   - понимание по датам прихода товара на склад;
   - согласование с клиентом даты отгрузки;
   - контроль приёма клиентом оборудования;
   - подписание документов на отгрузку;
   - назначение даты монтажа.


   6.  Создание облака для клиентов:


   - уточнение URL адреса с согласования клиента;
   - работа в nadmozg по созданию облака;
   - создание учётной записи технической поддержки в облаке клиента;
   - направление данных для входа клиенту и отправка обучающих материалов.


   7. Назначение обучения для новых клиентов и помощь в освоении приложения
   и программы:


   - согласование с клиентом даты и времени;
   - перенос в времени в случае необходимости;
   - сбор обратной связи после обучения;
   - направление записи обучения клиенту;
   - сбор необходимой информации для специалистов ТП (формат сервера, какое
   оборудование установлено, настроена ли интеграция, сколько точек
   проезда).


   8. Ежемесячное выставление счетов для клиентов на оплату:


   - 15 числа каждого месяца выставление счетов в 1С;
   - корректирование счетов в случае изменения суммы платежа;
   - контроль тех кто оплатил и кто ещё не заплатил.


   9. Общение с поставщиками на предмет закупаемого оборудования:


   - мониторинг цен у поставщика;
   - мониторинг наличия товара;
   - запрос счёта;
   - оплата счёта от поставщика;
   - контроль сроков отгрузки;
   - приёмка товара в том числе и дистанционно;
   - поиск нестандартного товара.


   10. Запрос счетов и внесение их в ПФ для оплаты:


   - запрос счёта в наиболее выгодной для нас организации в плане цены;
   - занесение счёта в план факт;
   - постановка задачи на оплату;
   - контроль оплаты.


   11. Сопровождение сервисного обслуживания:


   - контроль регламентных выездов по проведению сервисного обслуживания;
   - организация аварийных выездов в рамках 1-2 дней по договору;
   - закупка недостающих деталей при необходимости в рамках договора;
   - составление актов по выездам с подробным описанием сделанных работ.


   12. Сбор информации для отдела маркетинга:


   - сбор обратной связи с клиентов с их «хотелками» и пожеланиями;
   - подготовка информации для кейсов по клиентам;
   - размещений информации о нашей компании на сайтах поставщиков и
   клиентов;
   - договорённости о заключении партёнрских договоров с поставщиками;
   - запросы на благодарственные письма от клиентов.


   13. Ответственность за склад в БП Румянцево:


   - ведение склада в БП Румяцево;
   - перенос товара с склада в Румянцево в офис.


   14. Ведение и сопровождение сделок по апсейлам:


   - передача менеджерам ОП апсейлов по их клиентам;
   - ведение сделки по продаже оборудования клиентов, которые ни за кем не
   закреплены;
   - выявление потребности у клиентов.


   15. Ведение и сопровождение пилотных проектов:


   - назначение даты монтажа;
   - закупка и поставка оборудования;
   - постоянный мониторинг состояния оборудования по методу сбора обратной
   связи от заказчика;
   - выявление информации об окончании пилота и начале оплат.


   16. Ведение и сопровождение претензионных работ по клиентам, в том числе
   судебные дела при помощи юридического отдела:


   - работа с претензиями от клиентов;
   - снятие негатива;
   - непосредственное общение с юристами на предмет решения вопроса;
   - изучение постановлений суда и передачу в соответствующие инстанции;
   - возвращение уходящих клиентов;
   - работа с судебными претензиями с помощью юристов;
   - поиск и решение возникших проблем для сохранения лояльности клиентов.


   17. Отправка междугородних доставок при помощи ТК ( СДЭК,Деловые Линии):


   - заказ доставки или курьера;
   - отгрузка нашего оборудования на склад ТК;
   - мониторинг плановой даты отгрузки;
   - обратная связь от клиента принявшего отгрузку;
   - оплата счёта от ТК.


   18. Общение с подрядчиками:


   - поиск подрядчика для выполнения не профильных для нас работ;
   - согласование даты выезда подрядчиков для проведения сервисных работ по
   действующим клиентам;
   - мониторинг цен у подрядчика;
   - заключение договора в с подрядчиком;
   - контроль выезда и проведения работ у подрядчика.


   19. Заключение договоров с новыми и интересными нам организациями:


   - общение на предмет взаимовыгодных условий;
   - поиск наиболее ценного предложения;
   - обсуждение программы лояльности в адрес нашей компании.


   20. Работ с клиентами которые не закреплены за менеджером:


   - выявление потребностей;
   - сбор обратной связи;
   - решение текущих вопросов и предложение оптимального решения.
