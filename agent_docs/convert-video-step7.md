# Шаг 7 после convert-video-md: последовательность и чек-лист

Справочник для агента: как выполнять Шаг 7 (приведение MD к стандартам базы знаний) после конвертации видео. Шаг 7 выполняется **автоматически** для каждого созданного MD, без запроса пользователю.

**Шаблон статьи:** `agent_docs/Шаблон статьи базы знаний.md`

---

## Последовательность: 7.1 → 7.2 → 7.3 → 7.4

### 7.1. Исправление ошибок распознавания речи

1. Открыть MD и при необходимости сверяться с видео (`*-converted.mp4` / `*-converted.webm`).
2. Исправить типичные ошибки:
   - термины (PASS24, названия модулей, кнопок, разделов);
   - опечатки в интерфейсе («боковое меню», «правом верхнем углу» и т.п.);
   - числа, даты, искажённые технические термины;
   - пропущенные слова.
3. Проверить грамматику, пунктуацию, терминологию компании.

---

### 7.2. Структура и форматирование по шаблону

1. **Метаданные:** версия, даты, автор, тип документа, отдел.
2. **Целевая аудитория:** для кого, уровень, когда использовать.
3. **Краткое описание:** 2–3 предложения о назначении и содержании.
4. **Основной контент:** заголовки H2/H3, списки, пошаговые инструкции (что делать, где, результат, скриншот).
5. **Пошаговая инструкция** (для процедур): шаги с «Что делать», «Где», «Результат», ссылками на скриншоты.
6. **Контрольный чек-лист** (для процедур).
7. **История изменений:** таблица с версией 1.0 (дата, автор, «Первоначальная версия на основе видео»).

---

### 7.3. Дополнение (при необходимости)

- Проверить автоматически извлечённые скриншоты в `screenshots/` и их ссылки в MD.
- При необходимости: добавить скриншоты из видео, «Типичные ошибки и их решение», FAQ, «Связанные материалы».

---

### 7.4. Финальная проверка

- [ ] Ошибки распознавания исправлены

- [ ] Основной контент структурирован, есть пошаговая инструкция (если применимо)
- [ ] Скриншоты проверены, при необходимости добавлены
- [ ] Контрольный чек-лист (если применимо), история изменений
